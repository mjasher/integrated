loadBOMCSVData(filepath, data_cols=None, replace_missing=None, date_range=None)[source]¶

Load CSV data from BOM. Concatenating dates from Year, Month, Day columns.

Parameters:
  • filepath – Path to file including filename and extension
  • data_cols – (List) Data columns to import.
  • replace_missing – If numeric, replace all with given value. If string (‘ffill’ or ‘bfill’), replace with previous or next value See http://pandas.pydata.org/pandas-docs/stable/missing_data.html
  • date_range – array-like with two string values. Represents date range to extract. First value represents a start datetime, second value represents end datetime
